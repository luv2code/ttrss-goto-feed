# ttrss-goto-feed
Implement new keyboard shortcut for jumping to a feed.

Installation
------------
* Copy the *goto_feed* folder to your tt-rss *plugins.local/* folder.
* Go to your tt-rss Preference page
* Under *Plugins* section enable goto_feed plugin

Usage
------

Hit the keyboard shortcut `g g` and a dialog will popup. Type into the text box to auto complete a
category or feed name. Hit enter to navigate to the selected item.
